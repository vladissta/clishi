
## Описание функций 

+ Функции, в конце названия которых есть **_text_func**, находтся в `scripts/texts.R`
+ Функции, в конце названия которых есть **_calc**, находтся в `scripts/calculations.R`
+ Функции, в конце названия которых есть **_plot**, находтся в `scripts/plots.R`
+ Функции, в конце названия которых есть **_help**, находтся в `scripts/help_output.R`
