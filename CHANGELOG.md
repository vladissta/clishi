
## Изменения в server()

+ Многие if else заменены на switch   
+ Многие промежуточные переменные аля `n <- input$n` удалены, и теперь сразу используется `input$n`  
+ Добавил переменную `params_list()`, которая хранит все параметры распределений (norm, unif, exp)  
+ Добавил переменную `samples_values_simulated()` на замену `Xmat` - это датафрейм с колонками `experiment` (номер эксперимента) и `value` (значение выборки)
    + Во всех функциях этот датафрейм используется как аргумент `simulated_values_df = samples_values_simulated()`  
+ Основная симуляция теперь не возвращает список с множеством параметров, а возвращает датафрейм с результатами симуляции (**`df_from_sim`**)  
    + Все зависимости от параметров этого списка убраны и заменены на зависимости от параметров input  

## Описание функций 

+ Функции, в конце названия которых есть **_text_func**, находтся в `scripts/texts.R`  
Они нужны для вывода текстовой такой информации, как сводные статистики для одной выборки или как выглядят тестируемые гипотезы.

+ Функции, в конце названия которых есть **_calc**, находтся в `scripts/calculations.R`
Они нужны для проведения расчетов, например, симуляции выборок или рассчёта ДИ.

+ Функции, в конце названия которых есть **_plot**, находтся в `scripts/plots.R`
Они нужны для построения всех графиков

+ Функции, в конце названия которых есть **_help**, находтся в `scripts/help_output.R`
Они нужны для вывода графиков во вкладке HELP
